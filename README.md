# Jepretin

**Premium Browser-Based Photobooth**

Designed to capture moments with elegance. Jepretin delivers a high-quality, seamless photography experience directly in your browser, featuring a sophisticated white-themed aesthetic and client-side processing for absolute privacy.

---

### Key Features

- **Premium Interface:** A modern UI featuring mesh gradients, glass-morphism, and fluid animations.
- **Instant Capture:** Zero-latency camera feed with mirror mode and auto-countdown.
- **Advanced Editor:**
  - **Filters:** Aesthetic presets (B&W, Vintage, Sepia).
  - **Frames:** Curated minimal layouts (Polaroid, Film Strip).
  - **Tools:** Stickers, drawing, and multilingual text (ENG/KOR/CN).
- **Privacy First:** All image processing is handled client-side via the Canvas API. No photos are uploaded to any server.

### Tech Stack

Built with modern technologies for performance and scale:

- **React 18** (Vite)
- **Tailwind CSS**
- **Framer Motion**
- **Lucide Icons**

### Quick Start

```bash
# Clone
git clone https://github.com/Rezkydesyafa/jepretin-aja-photobooth-WebApp.git

# Install
npm install

# Run (localhost:5173)
npm run dev
```
